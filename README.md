# 기타 프로젝트

**평화촛불**

집회용 촛불 앱

* 상단 TextView는 내용과 위치(상단,하단) 설정 가능.
* Animation-list를 활용 , 20장의 촛불 이미지를 사용하여 타오르는 촛불을 표현.
* 촛불 세기 및 촛불이미지 변경 가능.
* 종이컵 이미지 중앙 구호 설정 기능
* Handler를 이용한 버튼 자동 숨김.
* SharedPreferences사용으로 개인 설정 유지 가능.
